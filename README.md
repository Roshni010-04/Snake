Snake Game | HTML • CSS • JavaScript

A classic Snake Game web application built using HTML, CSS, and JavaScript.  
This project demonstrates core front-end development skills, game logic implementation, and interactive user controls.

Project Overview

The Snake Game allows users to control a snake using keyboard arrow keys.  
As the snake eats food, it increases in length and the score updates in real time.  
The game also includes a scoreboard, high score tracking, and multiple difficulty levels.

Key Features

- Keyboard-based snake movement using **Arrow Keys**
- Dynamic snake growth on eating food
- Live Score Board
- High Score tracking
- Multiple difficulty levels:
  - Easy
  - Normal
  - Medium
  - Hard
- Smooth and responsive gameplay
- Clean and user-friendly UI

Controls

⬆️ Arrow Up – Move Up  
⬇️ Arrow Down – Move Down  
⬅️ Arrow Left – Move Left  
➡️ Arrow Right – Move Right  

Technologies Used

HTML– Game structure and layout  
CSS – Styling and visual design  
JavaScript– Game logic, movement control, speed, scoring system  

How to Run the Project

1. Download or clone the repository  
2. Open the project folder  
3. Run the `index.html` file in any modern web browser  
4. Start playing using arrow keys  

Learning Outcomes

- JavaScript logic building
- Event handling and DOM manipulation
- Game loop implementation
- Score and state management
- UI interaction handling
